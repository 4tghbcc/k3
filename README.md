Особенности реализации
✅ Зацикливание - после последнего изображения идет первое, перед первым - последнее
✅ Навигация - кнопки "вперед/назад" и точки для быстрого перехода
✅ Счетчик - показывает текущий номер и общее количество изображений
✅ Адаптивный дизайн - корректно отображается на всех устройствах
✅ Клавиатурная навигация - стрелки влево/вправо
✅ Плавные переходы - анимация смены изображений
✅ Современный дизайн - тени, градиенты, hover-эффекты
